# Roadmap: G&P Landing Page

## Overview

A single-phase delivery of the complete G&P landing page -- from foundation and design system through content sections, lead generation CTAs, legal compliance, and production hardening. One phase because the deliverable is one page: it either compels Geschaeftsfuehrer to act or it does not. There is no meaningful partial delivery.

**Phase Numbering:**
- Integer phases (1, 2, 3): Planned milestone work
- Decimal phases (2.1, 2.2): Urgent insertions (marked with INSERTED)

## Phase Details

### Phase 1: Complete Landing Page
**Goal**: Visitors land on a bold, provocative, fully functional G&P landing page that compels Geschaeftsfuehrer to act -- signing up for the Sch(B)rechstunde KI or contacting G&P directly
**Depends on**: Nothing (first phase)
**Requirements**: DESIGN-01, DESIGN-02, DESIGN-03, DESIGN-04, DESIGN-05, DESIGN-06, CONTENT-01, CONTENT-02, CONTENT-03, CONTENT-04, CONTENT-05, CONTENT-06, LEAD-01, LEAD-02, LEAD-03, LEGAL-01, LEGAL-02, LEGAL-03, TECH-01, TECH-02, TECH-03, TECH-04, TECH-05
**Success Criteria** (what must be TRUE):
  1. A visitor on mobile or desktop sees a bold, dark-themed page with dramatic typography, scroll animations, and real converted photos -- unmistakably G&P, not a generic consulting template
  2. A visitor can follow the full persuasion arc from hero (dramatic slogan + primary CTA) through problem agitation, three Themenschwerpunkte blocks, Sch(B)rechstunde KI section, to footer with links to v-und-s.de and neues-management.jimdofree.com
  3. A visitor can reach G&P via mailto CTAs distributed at multiple points throughout the page, including hero and Sch(B)rechstunde sections
  4. The page is legally compliant with Impressum link, Datenschutz link, and DSGVO cookie/CDN disclosure banner, and deploys correctly on GitHub Pages with Lighthouse mobile score >= 90
  5. German compound words render correctly with proper hyphenation, OG meta tags produce rich social previews when shared on LinkedIn, and the page loads in under 3 seconds on a 3G connection
**Plans**: 5 plans
**UI hint**: yes

Plans:
- [ ] 01-01-PLAN.md -- Foundation: Tailwind theme, image conversion pipeline, build tooling
- [ ] 01-02-PLAN.md -- Page skeleton: HTML head, nav bar, hero section, problem agitation
- [ ] 01-03-PLAN.md -- Content sections: pillars, credibility, Sch(B)rechstunde, CTA, footer, cookie banner
- [ ] 01-04-PLAN.md -- Interactivity: GSAP scroll animations, mobile nav, cookie banner JS
- [ ] 01-05-PLAN.md -- Production hardening: Tailwind CLI build, OG meta, performance validation
